‚úÖ Solution yang Diterapkan:

1. Fixed vercel.json

}

  • ‚úÖ Menghindari "Mixed Routing Properties" error

3. Backup Configuration

Created vercel.alternative.json jika diperlukan:

{
  "functions": {
    "app/[[...routes]].tsx": {
      "maxDuration": 10
    }
  },
  "rewrites": [
    {
      "source": "/api/(.*)",
      "destination": "/api/$1"
    },
    {
      "source": "/(.*)",
      "destination": "/index.html"
    }
  ]
}
